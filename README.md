# Movie Booking App 🎬

## Description
This is a React-based Movie Booking application where users can browse movies, view details, and book tickets by filling out a form. After booking, a confirmation screen displays the user details along with a randomly generated booking ID.

## Features
- View list of movies
- Movie details page
- Book tickets via form (Name, Email, Mobile)
- Booking confirmation with unique ID
- Navigation using React Router

## Technologies Used
- React JS
- JavaScript
- CSS
- React Router

## How to Run

1. Install dependencies:
   npm install

2. Start the app:
   npm start

3. Open in browser:
   http://localhost:3000
